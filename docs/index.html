WISG Module 4: Computational Pipeline for WGS Data

WISG Module 4: Computational Pipeline for WGS Data

2018-12-18

1 Introduction

This site contains course materials for WISG Module 4: Computational Pipeline for WGS Data, January 23-24, 2019. Data used is located in the github repository from which the site is built, as well as in the TOPMed analysis pipeline.

1.1 Software requirements

  • R 3.5.1
  • R packages
    • SeqArray
    • SeqVarTools
    • GENESIS
    • SNPRelate
    • TopmedPipeline
    • dplyr
    • ggplot2
    • GGally
    • survey
    • CompQuadForm
    • TxDb.Hsapiens.UCSC.hg19.knownGene

Installation instructions

install.packages(c("BiocManager", "remotes"), repos="https://cloud.r-project.org")

BiocManager::install(c("SeqArray", "SeqVarTools", "SNPRelate", "GENESIS", 
                       "survey", "CompQuadForm", "dplyr", "ggplot2", "GGally",
                       "TxDb.Hsapiens.UCSC.hg19.knownGene"))

remotes::install_github("UW-GAC/analysis_pipeline/TopmedPipeline")
